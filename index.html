Dots & Boxes Game

A two-player game in which players take turns connecting dots to complete boxes. A player can complete a box started by the other player. When a player completes a box, he/she gets another turn. The player with the most completed boxes wins.

Built with HTML, CSS, JavaScript, and jQuery in a one-week sprint.

Project Site GitHub

Il Foro Romano (The Roman Forum)

A forum for Italophiles to share their love for the Italian language and culture, organize local meetups, find language partners, and more. Users can view posted content without signing up, but must sign up and log in to post, comment on posts, and upvote or downvote posts.

Built with Ruby on Rails, PostgreSQL, HTML, CSS, and Bootstrap in a one-week sprint.

Project Site GitHub

Spark

Get introduced to new potential areas of interest through randomly generated TED Talks. Sign up and log in to see local Meetup groups in your area where you can further explore the topic introduced in the TED Talk.

Built in a team of three in a one-week sprint using MongoDB, Express, Node.js, HTML, CSS, Bootstrap, the YouTube API, the Meetup API, and Ajax.

Project Site GitHub

CaliApp

A crowd-sourced travel planning tool for those looking to visit or explore California. Users can see user-recommended destinations on a map and can log in to add destinations of their own and to save destinations to their profile.

Built in a team of two in a one-week sprint using MEAN stack (MongoDB, Express, Angular.js, Node.js), HTML, CSS, Bootstrap, and the Google Maps API.

Project Site GitHub
